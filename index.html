Under the Hood

VoiceAuthentix uses a hybrid CNN-BiLSTM architecture trained on the ASVspoof 2021 dataset containing over 600,000 genuine and spoofed utterances. The model excels at detecting TTS-generated speech (WaveNet, MelGAN, HiFi-GAN), voice conversion, and replay attacks.

Mel-spectrograms expose subtle artifacts in AI-generated audio — unnatural harmonic distributions, phase discontinuities, and frequency smearing — that are invisible to human listeners but detectable by deep learning.

# Mel-Spectrogram Extraction
def extract_mel(audio, sr=22050):
    y = librosa.effects.preemphasis(audio)
    mel = librosa.feature.melspectrogram(
        y=y, sr=sr,
        n_mels=128,
        n_fft=2048,
        hop_length=512
    )
    return librosa.power_to_db(
        mel, ref=np.max
    )

# Inference Pipeline
def classify(audio_chunk):
    mel = extract_mel(audio_chunk)
    mel = normalize(mel)
    tensor = torch.FloatTensor(mel)
    with torch.no_grad():
        logits = model(tensor)
        prob = torch.softmax(logits, dim=1)
    return {
        'real': prob[0][0].item(),
        'fake': prob[0][1].item()
    }
